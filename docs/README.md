# Документация Eltera

Вся документация по продукту, технической части, юридическим вопросам и исследованиям.

- `product/` — продуктовые решения: MVP, роадмап, флоу, тарифы
- `technical/` — архитектура, API, база данных, деплой
- `manus/` — промпты, задачи и журнал изменений для Manus
- `legal/` — юридические документы
- `research/` — исследования рынка и конкурентов
