# Роли и права доступа

| Роль | Описание | Права |
|---|---|---|
| Super Admin | Владелец аккаунта | Все |
| HR Admin | HR-директор | Управление оценками |
| Manager | Руководитель | Просмотр своей команды |
| Employee | Сотрудник | Прохождение оценок |
| Candidate | Кандидат | Прохождение тестов |
