Кандидаты
Вариант 1 · Одна строка — период + dropdown-фильтры в карточке
✓ Всё в одну строку, не переносится. Период — сегментированные кнопки, фильтры — компактные dropdown-кнопки с шевроном. Редкие фильтры скрыты за «+ ещё 4». Нужны dropdown-попапы при клике.
Вариант 2 · Inline-чипы — активные фильтры видны сразу
✓ Активные фильтры видны как теги — можно снять одним кликом. Неактивные фильтры — обычные чипы. Редкие скрыты за «+ Фильтры». При многих активных фильтрах строка может переполниться.
Вариант 3 · Collapsed — кнопка «Фильтры» + активные теги рядом ← рекомендую
HH.ru × Подходит ×
✓ Максимально компактно: одна строка всегда. Период — слева, кнопка «Фильтры» с бейджем количества активных — открывает панель/дропдаун. Активные фильтры показаны как теги рядом — можно снять. Если фильтров нет — теги не занимают место. Идеально для страниц с большим числом фильтров.
