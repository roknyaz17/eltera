# Eltera Assessment Intelligence

Репозиторий проекта Eltera — AI-платформы оценки кандидатов и сотрудников.

## Структура репозитория

| Папка | Содержимое |
|---|---|
| `web-app/` | Исходный код веб-приложения |
| `docs/` | Документация: продукт, техническая, юридическая, исследования |
| `brand/` | Брендбук, логотипы, цвета, шрифты |
| `assessment-library/` | Библиотека оценок: профессии, компетенции, вопросы, отчёты |
| `templates/` | Шаблоны: Excel, PDF, JSON, email |
| `outputs/` | Выгрузки и экспорты (не коммитятся) |
| `prompts/` | Промпты для AI и Manus |
| `data/` | Тестовые и sample-данные |

## Быстрый старт

```bash
cd web-app
npm install
npm start
```
