# Brand

Все бренд-материалы Eltera.

| Папка | Содержимое |
|---|---|
| `brandbook/` | Файлы брендбука (PDF, Figma-ссылки, экспорты) |
| `logos/` | Логотипы во всех форматах: SVG, PNG, тёмная/светлая версии |
| `colors/` | Цветовые токены, палитры, ASE/Sketch-файлы |
| `fonts/` | Шрифты Inter, JetBrains Mono |
| `references/` | Референсы, mood board, вдохновение |
