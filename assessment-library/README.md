# Assessment Library

Библиотека оценочных материалов Eltera.

| Папка | Содержимое |
|---|---|
| `professions/` | Профили должностей и ролей |
| `competencies/` | Компетенции по категориям |
| `questions/` | Банк вопросов по ролям |
| `reports/` | Шаблоны и структуры отчётов |
| `templates/` | Шаблоны оценочных форм |
